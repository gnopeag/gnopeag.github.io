Github Copilot Free 现已推出！

2024-12-19

官方解释：About GitHub Copilot Free

一、Github Copilot 概述

GitHub Copilot 是一款革命性的 AI 编码助手，旨在加速开发流程，让开发者能够更高效地编写代码。通过自动补全代码片段、提出优化建议和生成文档，它使得开发人员可以将更多的时间投入到创新和问题解决上。在 AI 辅助编程的领域中，GitHub Copilot 已成为不可或缺的工具，为无数开发者提供了强大的支持。然而，订阅费用一直是部分用户考虑的因素。现在，为了让更多开发者体验到这一先进的技术，GitHub 推出了全新的 GitHub Copilot Free 版本。

二、功能概览

GitHub Copilot Free 提供了对核心编码帮助功能的有限访问，无需订阅即可享受：

  1. 多 IDE 支持：兼容 Visual Studio Code、Visual Studio、JetBrains IDEs、Vim/Neovim、Xcode 和 Azure Data Studio，实现跨平台的代码完成。
  2. Copilot 聊天：集成于 Visual Studio Code、Visual Studio、JetBrains IDE 和 GitHub.com，提供智能对话式编码辅助。
  3. 安全编码保障：阻止与公共代码库直接匹配的代码建议，确保代码的独特性和安全性。
  4. 先进模型驱动：基于 Claude 3.5 Sonnet 模型，提供高质量的代码建议和服务。

三、使用限制

尽管 GitHub Copilot Free 提供了诸多便利，但它并非没有限制：

  1. 代码完成次数：每月限 2000 次代码完成机会。
  2. 聊天消息数量：每月最多可发送 50 条聊天消息，涵盖标准聊天及多文件编辑会话。
    对于部分重度使用者来说，这些限制可能意味着需要考虑升级至付费版本以获得无限制的使用体验。

四、VSCode 快速上手

只需简单几步就能开始使用这项服务：

  1. 打开 VSCode 并安装 GitHub Copilot 扩展。
  2. 启动一个 Copilot 会话，点击连接到 GitHub 进行登录授权。
  3. 完成授权后返回 VSCode，便可以立即开始利用 Copilot 提供的智能编码帮助了。👍
